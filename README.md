# Informatique et Intelligence Artificielle — IIA2106

**Enseignant:** Emmanuel Noutahi, PhD

---

## À Propos du Cours

Ce cours vous apprend à utiliser l'intelligence artificielle dans un contexte médical. Vous allez manipuler des données biomédicales, analyser des images médicales avec l'IA, et développer une réflexion critique sur l'utilisation responsable de ces technologies.

**Prérequis:** Aucune expérience en programmation requise.

**Plateforme:** Google Colab (gratuit)

**Langue:** Français

---

## Cours

### Déjà Complétés
1. **Introduction à Python** - Bases du langage, structures de données
2. **Analyse de Données Biomédicales** - Pandas, visualisation, ML
3. **PyTorch Fundamentals** - Réseaux de neurones, applications médicales
4. **Classification de Texte Médical** - NLP, classification multi-labels

### Prochains Cours
5. **Analyse de Radiographies Thoraciques** - Détection de pathologies avec TorchXRayVision
6. **Segmentation d'Images Médicales** - Segmentation interactive avec MedSAM
7. **Éthique de l'IA en Médecine** - Biais, responsabilité, déploiement responsable
8. **Démonstration nnU-Net** - Segmentation automatique d'images médicales

## Comment Utiliser ce Cours

### 1. Ouvrir un Notebook

- Cliquez sur le badge "Open in Colab" en haut de chaque notebook
- Ou allez sur [colab.research.google.com](https://colab.research.google.com) → Fichier → Ouvrir depuis GitHub → `maclandrol/cours-ia-med`

### 2. Sauvegarder Votre Travail

- Fichier → Enregistrer une copie dans Drive
- Le notebook sera sauvegardé dans votre Google Drive

### 3. Exécuter le Code

- Cliquez sur ▶️ ou appuyez sur `Shift+Enter`
- Suivez les instructions dans chaque cellule

### 4. Obtenir de l'Aide

Consultez le fichier `AIDE_COLAB.md` pour résoudre les problèmes courants.

## Objectifs d'Apprentissage

À la fin de ce cours, vous serez capable de:

- Analyser des données biomédicales avec Python
